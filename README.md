# DirtyCafeSales---DataCleaningTraining
# ☕ Dirty Cafe Sales Data Cleaning & Visualization

## 📌 Project Overview

This project focuses on cleaning and analyzing a real-world dirty café sales dataset using Python. The dataset contains missing values, incorrect entries, duplicate records, and inconsistent data types, making it ideal for practicing data preprocessing and Exploratory Data Analysis (EDA).

After cleaning the data, various visualizations are created to identify customer purchasing patterns, revenue trends, and business insights.

---

## 🎯 Objectives

* Clean and preprocess the dataset
* Handle missing values and invalid entries
* Detect and remove duplicates
* Convert data into appropriate formats
* Perform Exploratory Data Analysis (EDA)
* Create meaningful visualizations
* Generate business insights from the cleaned data

---

## 📂 Dataset Information

**Dataset Name:** Dirty Cafe Sales Dataset

### Columns

* Transaction ID
* Item
* Quantity
* Price Per Unit
* Total Spent
* Payment Method
* Location
* Transaction Date

* Python

## 📊 Data Cleaning Process

The following preprocessing steps were performed:

* Removed duplicate records
* Replaced invalid values such as `ERROR` and `UNKNOWN`
* Handled missing values
* Converted numerical columns into proper numeric data types
* Converted transaction dates into datetime format
* Checked for outliers
* Prepared the dataset for visualization
